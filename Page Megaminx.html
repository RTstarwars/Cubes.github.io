The Megaminx


This cube is a dodecahedron which means that it has 12 sides. In some ways it is quite similar to the 3x3 rubik's cube so I suggest you learn that first.

1. The first thing that you have to do is choose a colour.
2. Then solve that colour like you would solve the first layer of a 3x3.
